# Solution Plan Rules

Rules for phased documentation plans and plan review.

## Chain

- **plan-engineer** — Creates phased documentation plans with RAID logs and clear definitions of done. Use when breaking doc/article work into executable phases.
- **plan-reviewer** — Reviews plans for atomicity, completeness, feasibility. Use after the plan-engineer produces a plan.

## When to use which

- Creating or updating a documentation plan → **plan-engineer**
- Validating a plan before execution → **plan-reviewer**
